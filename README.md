# Node.js Base

This repository provides a base project for Node.js applications, ready to be cloned and used as a starting point for new projects.

## Dependencies

```bash
"dependencies": {
    "cors": "^2.8.5",
    "express": "^5.2.1",
    "helmet": "^8.1.0"
  },
"devDependencies": {
    "@types/cors": "^2.8.19",
    "@types/express": "^5.0.6",
    "@types/node": "^25.0.3",
    "ts-node": "^10.9.2",
    "tsx": "^4.21.0",
    "typescript": "^5.9.3"
  }
```
## Getting started

1. **Clone the repository:**

  ```bash
    git clone https://github.com/guilhermep3/node-base
  ```

2. **Install the dependencies:**

  ```bash
    npm install
  ```

3. **Start the project:**

  ```bash
    npm run dev
  ```
